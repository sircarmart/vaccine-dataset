# Vaccine Tweets Dataset

This repository contains the dataset and code used for our paper *"The Shifting Landscape of Vaccine Discourse: Insights From a Decade of Pre- to Post-COVID-19 Vaccine Posts on Social Media"*.

The ```dataset/vaccines_tweet_ids``` folder contains ```.zip``` files with tweet ids of vaccine tweets from years 2013-2022. The ```dataset/control_medical_tweet_ids``` contains the tweet IDs of tweets with medical terms.
